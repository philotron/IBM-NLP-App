About

The Staffing Advisor web application serves as an example showing how to use the developed matching engine for assisting in the creation of SOWs.

It guides the user in the SOW creation process by going through the following stages:
1. Input of general job title and job description.
2. Selection of skills based on the matching results.
3. Specification of requirements based on selected skills and matching results.

In the end, the user has the option to give feedback about the service.
